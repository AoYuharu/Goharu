# TableHelper TUI 快速启动指南

## 启动步骤

1. **启动 TUI**
   ```bash
   python run_tui.py
   ```

2. **等待 Gateway 启动**
   - 状态栏会显示 "Waiting for Gateway..."
   - 几秒后会显示 "Gateway ready - You can start chatting!"

3. **开始聊天**
   - 在左下角的输入框中输入消息
   - 按 **Enter** 发送
   - 如果看不到输入框，按 **Tab** 键切换焦点

## 快捷键

- **Ctrl+C** - 退出 TUI
- **Ctrl+L** - 清空聊天历史
- **Tab** - 在组件间切换焦点
- **Enter** - 发送消息

## 常见问题

### 1. 看不到输入框？

**解决方法：**
- 按 **Tab** 键切换焦点到输入框
- 调整终端窗口大小（至少 80x24）
- 确保终端支持全屏应用

### 2. 无法输入内容？

**解决方法：**
- 确认输入框有焦点（应该有光标闪烁）
- 按 **Tab** 键切换到输入框
- 点击输入框区域（如果终端支持鼠标）

### 3. Gateway 启动失败？

**解决方法：**
- 检查日志：`logs/tui_gateway_crash.log`
- 运行诊断：`python diagnose_tui.py`
- 检查依赖：`pip install textual`

## 使用技巧

1. **调整终端大小** - 推荐至少 100x30
2. **使用现代终端** - Windows Terminal, iTerm2
3. **按 Tab 切换焦点** - 在输入框和其他组件间切换
4. **查看右侧面板** - 实时显示工具调用过程
